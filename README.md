## Project

Rebels is a self-hosted social platform with Rebels branding applied across the web UI, logos, mascots, and defaults.

## Local Development

Use the setup guide:

- `REBELS_SETUP.md`

## Core Stack

- Ruby on Rails
- PostgreSQL
- Redis + Sidekiq
- Node.js + React
